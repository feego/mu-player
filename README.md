# mu-player

### Features

  Play music from vk.com and soundcloud.com using Music Player Daemon(http://www.musicpd.org/)
  Music browser is powered by Last.FM.

## Install

# on linux
```$ sudo apt-get install mpd```

# on macos
```$ brew install mpd```
